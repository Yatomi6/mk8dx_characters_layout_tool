These DLLs are the minimal Switch Toolbox dependencies needed to edit BFRES files from Python via pythonnet:

- Syroot.NintenTools.NSW.Bfres.dll
- Syroot.Maths.dll
- Syroot.BinaryData.dll

They come from Switch Toolbox (https://github.com/KillzXGaming/Switch-Toolbox) and the Syroot NintenTools project.
